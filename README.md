# virtual environments

```python
python -m venv env

# activate
source env/bin/activate
.\env\Scripts\activate
deactivate


# install django
pip install django

# requirements.txt
pip freeze > .\requirements.txt

# install > requirements.txt
pip install -r requirements.txt

```
